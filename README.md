# 🛒 One Class Shop

One Class Shop is a Django-based e-commerce storefront for browsing products, adding items to a cart, placing orders, and reviewing order history. The project includes authentication, product pages, checkout flow, and an admin dashboard for managing products and orders.

---

## ✨ Features

- Product catalog with image previews, pricing, and stock display  
- Dedicated product detail pages  
- Cart management for adding, updating, and removing items  
- Login and registration pages for customer accounts  
- Order placement and confirmation screens  
- Order history page for signed-in users  
- Django admin support for managing products and orders  

---

## 🛠 Tech Stack

- Python 3  
- Django  
- SQLite (default)  
- Bootstrap 5  

---

## 📁 Project Structure

- `mysite/` → Django project settings and root URLs  
- `prouct/` → Main storefront app  
- `templates/` → UI templates (HTML pages)  
- `db.sqlite3` → Local development database  
- `manage.py` → Django management entry point
